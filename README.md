## `test-site`

Одностраничный сайт для предварительной регистрации на фестиваль.

### Файловая структура

В репозитории файлы хранятся в следующих директориях:

* `static/` &mdash; статические файлы сайта
* `src/` &mdash; файлы пакета Python-FastAPI сервера

### Локальный запуск сервера

В качестве предварительной настройки вам необходимо установить [uv](https://docs.astral.sh/uv/getting-started/installation/) &mdash; программу для управления зависимостями сервера.

После установки откройте терминал (в VS Code можно нажать клавишесочетание <code>ctrl+`</code>) и выполните следующую команду:

    uv sync

Это установит зависимости (все нужные библиотеки и Python, если нужно).

Теперь для запуска сервера выполните одно из следующих действий:

*   Выполните в терминале прямую команду запуска:

        uv run test-site

*   Используйте панель `Run and Debug` (`ctrl+shift+D`) в VS Code. Выберите профиль "Запуск сервера" и нажмите на зелёную кнопку `Start debugging (F5)`.

После запуска сайт можно открыть по ссылке `http://localhost:8000`.

### Запуск в Docker

Docker предоставляет полную изоляцию для сервера. В продакшн среде запуск сервера обязательно необходимо будет производить через Docker.

Перед запуском требуется установить Docker. Воспользуйтесь одной из ссылок ниже:

*   [Docker Engine](https://docs.docker.com/engine/install) &mdash; минимальная установка движка Docker, достаточная для запуска сервера
*   [Docker Desktop](https://docs.docker.com/desktop/) &mdash; графический интерфейс для Docker с дополнительными функциями для управления контейнерами

После установки Docker и `uv` (см. выше) выполните следующие две команды:

    # Синхронизация зависимостей
    uv sync --dev 

    # Запуск контейнера
    uv run inv up

    # Можно указать порт, если нужен порт, отличный от 8000
    uv run inv up --port 80

После запуска сайт можно открыть по ссылке `http://localhost:8000` (порт, соответственно, следует заменить на указанное число).
