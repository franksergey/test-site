# Структура директории static/

Директория `static/` содержит ресурсы сайта в различных папках. Поскольку сайт должен содержать несколько версий и иметь возможность к горячему переключению между ними в назначенный срок, была реализована система переключения между ними.

## 1-я версия сайта

Все файлы первой версии содержатся в директории `static/1/`. В этой версии на сервере доступны только файлы из этой папки, внешние файлы (видео, черновые файлы разметки) невозможно прочитать.

На этой версии работает эндпоинт для отправки email адресов: `POST /emails`.

## Другие версии

Сейчас реализована только одна версия сайта. Для будущих версий необходимо будет содать новые (новую) папки. При этом можно будет настроить автоматический скрипт для обновления версии на сервере в назначенное время.

Текущая сделанная на коленке система версионирования поддерживает тестирование различных версий при разработке. Для подробностей или информации о добавлении новых версий обратитесь к @soucelover.
